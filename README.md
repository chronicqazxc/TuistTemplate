# TuistTemplate

This is the Tuist template example for iOS projects.
[Tuist](https://tuist.io) is to tool to bootstrap, maintain, and interact with Xcode projects.

## How to use
1. Install tuist, welcome check my post [use-tuist-to-bootstrap-reproducible-project](https://wayneyuhanhsiao.wordpress.com/2021/10/08/use-tuist-to-bootstrap-reproducible-project/) or check the office [documentation](https://docs.tuist.io/tutorial/get-started#install)
2. Download the Tuist directory.
3. Create a empty directory for your project.
4. Put the Tuist directory under your project directory.
5. in the terminal.
6. `cd project_directory`
7. `tuist scaffold [wayneUIKit13Below | wayneUIKit13Above | wayneSwiftUI] --name your_project_name`
8. `tuist dependencies fetch`
9. `tuist generate --open`
10. The project will created base on the template you choose.

You may check my post [use-tuist-scaffold-to-create-reproducible-project](https://wayneyuhanhsiao.wordpress.com/2021/10/11/use-tuist-scaffold-to-create-reproducible-project/) for further introduction of my template.
